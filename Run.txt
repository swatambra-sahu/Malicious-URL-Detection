Extract Web_Extension_API.zip

backend/ (Flask API + ML model)
extension/ (Chrome extension)
 Step 2: Setup Backend (ML + API)
👉 Open terminal inside backend folder
🔸 Create virtual environment
python3 -m venv env
Activate:
Mac/Linux:
source env/bin/activate
Windows:
env/Scripts/activate
🔸 Install dependencies
pip install flask scikit-learn pandas numpy xgboost

You should have:
model.pkl

Run your training script first (MaliciousURLDetection.py)
Save model using:

python app.py

Running on http://127.0.0.1:5000
